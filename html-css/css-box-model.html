CSS Box Model The CSS Box Model is a fundamental concept in web design that describes how elements are structured and displayed in a rectangular box on a webpage. It determines the element's size, padding, borders, and margins, and how these interact with each other to affect layout and spacing. Box Model Structure Every HTML element can be thought of as a rectangular box with the following layers (from the inside out): Content: The actual content inside the element, such as text or images. Padding: Space between the content and the border. It clears the area around the content inside the element. Border: A line wrapping around the padding (if any) and the content. Margin: Space between the element's border and adjacent elements. Visualization +-------------------------------+ | Margin | +-------------------------------+ | Border | +-------------------------------+ | Padding | +-------------------------------+ | Content | +-------------------------------+ Properties of the Box Model 1. Content The width and height of the content area. Controlled using: width: 200px; height: 100px; 2. Padding Space inside the element between the content and the border. Controlled using: padding: 10px; /* Equal padding on all sides */ padding: 10px 20px; /* Vertical | Horizontal */ padding: 10px 15px 20px 25px; /* Top | Right | Bottom | Left */ 3. Border Defines the thickness, style, and color of the element's border. Controlled using: border: 2px solid black; 4. Margin Space outside the border to separate the element from others. Controlled using: margin: 10px; /* Equal margin on all sides */ margin: 10px 20px; /* Vertical | Horizontal */ margin: 10px 15px 20px 25px; /* Top | Right | Bottom | Left */ Example Code CSS Box Model

Box Model Calculation The total width and total height of an element include content, padding, border, and margin: Formula: Total Width = width + padding-left + padding-right + border-left + border-right + margin-left + margin-right Total Height = height + padding-top + padding-bottom + border-top + border-bottom + margin-top + margin-bottom Example Calculation (from the code above): Content width = 200px Padding = 20px (on each side) Border = 5px (on each side) Margin = 15px (on each side) Total width = 200 + 20 + 20 + 5 + 5 + 15 + 15 = 280px Box-Sizing Property The box-sizing property changes how the total size of the box is calculated. Values: content-box (Default): Width and height include only the content. Padding and border are added to the size. Example: box-sizing: content-box; border-box: Width and height include content, padding, and border (but not margin). Example: box-sizing: border-box; Example: .box { width: 200px; height: 100px; padding: 20px; border: 5px solid blue; box-sizing: border-box; /* Includes padding and border in total width/height */ } Practical Tips Use border-box for consistent layouts. Test designs with various padding, margins, and borders. Combine with responsive design techniques like media queries.
