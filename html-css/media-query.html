What is a Media Query? A media query is a CSS technique introduced in CSS3 that allows you to apply specific styles to a webpage based on the characteristics of the user's device, such as screen size, resolution, orientation, and more. Media queries are fundamental for building responsive designs, ensuring that your website looks good on all devices, from desktops to smartphones. Syntax of Media Query @media media_type (media_feature) { /* CSS rules here */ } @media: Defines the media query block. media_type: Specifies the type of device, such as screen, print, or all. (Optional; defaults to all.) media_feature: Specifies a condition, such as screen width, height, orientation, etc. CSS Rules: The styles to apply when the condition is true. Common Media Features Media Feature Description Example min-width Sets styles if the viewport width is at least X pixels @media (min-width: 600px) max-width Sets styles if the viewport width is at most X pixels @media (max-width: 600px) min-height Sets styles if the viewport height is at least X pixels @media (min-height: 400px) orientation Matches device orientation (landscape or portrait) @media (orientation: portrait) resolution M atches screen resolution (DPI or DPCM) @media (min-resolution: 300dpi) Example Media Queries 1. How Media Queries Work Evaluate Conditions: The browser checks the device's properties (e.g., screen width or orientation). Apply Styles: If the conditions match, the styles inside the media query block are applied. Use Cases of Media Queries Responsive Web Design: Create layouts that adjust to different screen sizes (mobile, tablet, desktop). Optimized Print Styles: Apply specific styles when printing documents. Example: @media print { body { font-size: 12pt; color: black; } } Dark Mode: Adapt styles based on user preferences. Example: @media (prefers-color-scheme: dark) { body { background-color: black; color: white; } } Best Practices Use mobile-first design: Start with styles for smaller screens and add media queries for larger screens. Example: body { font-size: 16px; /* Default for mobile */ } @media (min-width: 768px) { body { font-size: 18px; /* Larger screens */ } } Combine multiple conditions using and or or. Test on multiple devices and browsers.
